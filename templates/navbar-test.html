Mobile Navbar Test

✅ Mobile View Active: You should see a hamburger menu button in the navbar. Click it to test the collapsible menu functionality. ⚠️ Desktop View: Resize your browser window to less than 768px width or use mobile device emulation to test the mobile navbar.

Navbar Mobile Test Page

Testing Checklist:
  1. Hamburger Menu: Should appear on screens ≤ 768px
  2. Menu Toggle: Clicking hamburger should open/close the menu
  3. No Overlapping: Logo and hamburger button should not overlap
  4. Menu Items: Should stack vertically in mobile menu
  5. Click to Close: Clicking menu items should close the mobile menu
  6. Outside Click: Clicking outside menu should close it
  7. No Horizontal Scroll: Page should not scroll horizontally
  8. Text Readability: All text should be readable at small sizes
